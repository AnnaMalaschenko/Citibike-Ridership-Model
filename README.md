# Citibike Ridership Model

## Project Purpose
Identify what drives daily ridership to inform bike availability and staffing.

Query datasets with SQL to combine a table of 50 million individual bike trips with a table of daily weather observations to create a dataset that can be used to build a model.

## Folder Descriptions
code - notebooks showcasing work
data - exported CSV of joined trips and weather tables
queries - SQL queries that were run
docs - supporting documents
